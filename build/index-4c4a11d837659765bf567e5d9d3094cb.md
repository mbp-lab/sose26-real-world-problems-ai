# Welcome to the course!

Over the coming months, we'll dive into real-world AI problems, with examples that inspired each chapter's theme. This Jupyter Notebook is designed for the tutorial but it will also connect to the seminar of the same name. 

The goal of the tutorial is to learn as much as possible in a lively, collaborative environment!
Each chapter begins with an introduction, followed by notebooks that you'll need to download, complete and submit via Moodle. After the deadline, class notes will be released for further study, along with optional materials for those who want to go the extra mile.
