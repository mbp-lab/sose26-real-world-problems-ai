You started with exploring how distances behave in higher dimensions. What you would have observed is that distances tend to increase as the number of dimensions grows. This effect appears across distance metrics. While some metrics may increase more gradually than others, this is a phenomenon that no metric can entirely escape. That contributes to what is known as the curse of dimensionality. As distances between points increase, it becomes harder to distinguish between them. Intuitively, points become like isolated dots scattered in a vast space, where everything is far apart and differences become less meaningful. 

When we speak of the dimensionality of a dataset, we are referring to the number of features or variables used to describe each data point. In machine learning, this has important implications: as dimensionality increases, models often require more data to generalize well and distance-based methods (such as nearest neighbors) can become less effective because all points start to appear similarly distant.
